  • Delete every colonized planet
  • Move each homeworld to a new random position
  • Reset buildings, resources, and research back to a fresh start
  • Clear every stationed ship and in-flight fleet movement
  • Delete all messages

User accounts, usernames, and passwords are not affected — only game progress.
